# ♠️ Royal Blackjack

A premium Vegas-style Blackjack game built with **HTML, CSS, and JavaScript**.

Play here: https://carljooohn.github.io/BlackJack/

## 🎮 About

Royal Blackjack is a browser-based casino card game featuring a 4-deck shoe, betting chips, side bets, player stats, leaderboard tracking, and smooth casino-style animations.

## ✨ Features

### Gameplay
- Standard Blackjack rules
- Dealer stands on 17
- Blackjack pays 3:2
- Hit, Stand, Double Down, and Split
- Insurance when dealer shows Ace
- Push/tie handling
- Soft Ace scoring
- 4-deck shoe with reshuffle system
- Split-hand support

### Betting
- Starting balance of $1,000
- Chip-based betting
- Clear, redo, max bet, and reset options
- Persistent balance using browser storage

### Side Bets
- **Perfect Pair**
  - Mixed Pair: 6:1
  - Coloured Pair: 12:1
  - Perfect Pair: 25:1

- **21+3**
  - Flush: 5:1
  - Straight: 10:1
  - Three of a Kind: 30:1
  - Straight Flush: 40:1
  - Suited Trips: 100:1

### Visuals
- Vegas-style dark casino theme
- Animated card dealing
- Card flip effects
- Neon glow UI
- Chip animations
- Particle background
- Win and blackjack sound effects

### Player Progress
- Wins and losses tracking
- Current streak
- Best win
- Total profit
- Local leaderboard
- Player name support

### Controls
Keyboard shortcuts:

| Key | Action |
|---|---|
| `H` | Hit |
| `S` | Stand |
| `D` | Double Down |
| `P` | Split |
